# Flashcard App

A self-contained browser-based flashcard study tool. No server, no install, no internet required — open the HTML file and start studying.

Built initially for an archaeology course, but the structure works for any subject.

![HTML](https://img.shields.io/badge/HTML-E34F26?style=flat-square&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat-square&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

---

## Features

- Chapter selection grid — navigate directly to any chapter
- Flip animation with front/back card reversibility
- Tag filter chips for targeted review by topic
- "Know it / Still learning" ratings with persistent progress tracking
- Stats and progress bars per chapter
- Completion screen when a chapter is finished

---

## How to use

1. Download or clone the repo
2. Open `ANTH103_Flashcards.html` in any browser
3. Select a chapter and start flipping

No dependencies. No build step. Works offline.
